prototype sample

string, date객체의 prototype을 확장하여 편의를 제공하는 샘플로 더욱 다양한 prototype을 사용할수 있음.

string prototype

  • trim : s.trim(' my name is jun-hee,hong ');
  • lPad : s1.lPad('0', 3);
  • rPad : s1.rPad('0', 3);
  • toDate : s1.toDate('%q');
  • toXml : s1.toXml();

date prototype

  • add : date.add('D', 1);
  • compareTo : date1.compareTo(date2)
  • equals : date2.equals(date2)
  • format : date.format('%x')
  • clone : date.clone()
