Dom Control

dom의 기본 기능을 확장하고 dom selector를 통해 다수의 dom 객체를 동시에 제어할 수 있는 샘플

하나 이상의 dom을 selctor나 filter를 이용해 선택하여

show/hide 같은 기본 설정을 변경하거나 css style을 변경할 수 있습니다.

가) DOM의 id로 DOM 을 검색하여 제어 하는 구문

    $E('divContainer').setStyle('background-color', 'lightgreen');

나) DOM의 selector로 DOM 을 검색하여 제어 하는 구문

    $S('#divContainer .color').setStyle('background-color', 'orange');
    $S('#divContainer .color.class2').setStyle('background-color', 'inherit');

다) DOM의 selector와 filter를 이용하여 제어 하는 구문

    $S('#divContainer li').filter('.hide').hide();

라) DOM의 selector 메소드 체인을 이용하여 제어 하는 구문

    Rui.select('#divContainer input').setStyle('width', '200px').disable();
    $E('divContainer').select('input[name=input1]').setValue('Hello World').setStyle('width', '300px').enable();

* $S와 SE 메소드는 Rui.select와 Rui.get의 축약형 메소드로 제공되고 있다.
