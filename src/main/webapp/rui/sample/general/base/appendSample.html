HTML Page Append (Ajax)

AJAX를 이용하여 제공된 URL의 HTML 페이지를 읽어서 원하는 위치에 붙이는 기능

페이지 안에서 AJAX를 이용해 다른 페이지의 HTML을 읽어와 원하는 DOM 하위에 붙여 표현할 수 있습니다.

읽혀진 HTML 안의 script등도 자동 실행되며, RichUI의 각종 기능도 정상 동작합니다.

이 기능을 초기 페이지 로딩시에 사용할 경우 성능이 저하가 발생할 수 있으며, Memory Leak 등을 주의해야 합니다.
