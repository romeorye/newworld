Event Listener

이벤트 리스터(Event Listener)를 사용하여 간편하게 이벤트를 생성하고 제어할 수 있습니다.

HTML Dom객체의 이벤트(Event)를 제어합니다. 각 이벤트는 개별적 혹은 연속적으로 수행될 수 있습니다.

1.click => Dom을 클릭하면 dd-demo-2-2객체와 dd-demo-2의 mousedown 이벤트가 발생합니다.

2.click => Dom을 클릭하면 dd-demo-2의 mousedown 이벤트가 발생합니다.

3.click => Dom을 클릭하면 dd-demo-3의 mousedown 이벤트가 발생합니다.
