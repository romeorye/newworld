Sample

DevOn RichUI는 순수 HTML 표준 기술을 사용해 기존 Plug-in기반 X-Internet 도구들의 기능들을 제공하는 B2B용 Cross Browsing/Platform Rich UI 솔루션입니다.

RichUI를 사용하기 전에 먼저 몇가지 RichUI의 특징들을 살펴본 후 "Hello RichUI"를 출력해보도록 하겠습니다.

  • javascript 기반의 객체지향(OOP) 구조로 구현되어 있습니다.
  • javascript 기반으로 기존 Plug-in기반의 RIA 제품 등과 비교해 유사한 기능과 성능을 제공합니다.
  • 웹 표준 기술을 사용하므로 이를 따르는 모든 브라우저들에서 동작합니다. FireFox, Chrome, Safari 등.
  • 국내 환경에서 가장 많이 사용되는 Internet Explorer 6, 7, 8 을 지원합니다. 그 이후 출시된 Internet Explorer 10과 같은 최신 버전들도 출시 일정에 맞추어 테스트 되고 있습니다.
  • 현재는 PC 버전만 제공되고 있으나 모바일 등 다른 플랫폼으로 확장이 가능한 구조입니다.

그럼 Hello RichUI를 시작해 보도록 하겠습니다.
