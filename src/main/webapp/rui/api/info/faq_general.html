General

  • 디자인 적용 방법은?
    디자인 가이드 문서를 받아서 적용한다.
  • context path가 바뀌면 어떻게 하나요?
    rui_config.js의 contextPath값만 변경하면 된다.
    var contextPath = '/test';
  • 전자 정부 프레임웤과 연동하려면 어떻게 해야 하나요? 
    UI쪽은 서버의 프로그래밍 언어에 대해서 구분하지 않는다. 서버와 주고 받는 데이터는 json이나 xml의 RUI 표준 포멧에 맞춰 있으면 구동되는데 문제가 없다.
다만 서버 모듈은 DevOn과 연동 작업만 기본으로 되어 있으므로 다른 프레임웤에 대해서는 프로젝트에서 가이드를 받아 직접 구현해야 한다.
현재 전자 정부 프레임웤과의 연동은 RUI에 기본으로 사용할 수 있게 작업중이다.
  • firefox에서는 정상인데 ie에서만 에러가 발생하는데요? 
    RUI 프레임웤은 기본적으로 4대 브라우져(IE, firefox, chrome, safari)에 대해서는 기본적으로 테스트를 진행한다. 
물론 브라우져의 특이한 이슈사항은 계속 발생할 수 있으나 일반적인 문제는 대부분 해결되어 있으므로 크로스 브라우져의 
테스트 진행시 아래의 최소한의 기본 체크를 한 후에 해결이 안될 경우 지원 요청을 한다.
1. fireFox에서는 ','가 추가되어도 에러로 인식하지 않는다. 전체 소스에서 특정 화면의 경우에는 대부분 이문제이다.
2. 표준 script를 사용하고 있는지 확인한다.
  • 엑셀 다운로드는 어떻게 하나요? 
    LGridPanelExt 플러그인을 이용하면 화면의 그리드의 내용을 그대로 csv 파일로 저장할 수 있습니다. 
웹 표준을 따라 처리되어야 하기때문에 서버의 리소스를 이용하지만 출력된 그리드의 내용을 그대로 또는 사용자가 원하는 
방식으로 변환하여 저장이 가능합니다.
  • 암호화 모듈과 연계가 가능한가요? 
    현재 XecureWeb 모듈과 연계되어 있습니다. 그냥 XecureWeb을 화면에 탑재하는것만으로 바로 작동되게 되어있습니다. 
그외에 추가로 다른 암호화 모듈과 연계를 원할 경우 지원을 통해서 탑재가 가능합니다.
  • 차트는 존재하나요? 
    현재 차트는 존재하지 않습니다. 2012년에 개발 계획에 있습니다. 현재는 다른 차트를 프로젝트에서 직접 구매하여 연동하셔야 합니다.
  • 리포트는 존재하나요? 
    리포트는 존재하지 않습니다. 차후 계획에도 아직 반영되어 있지 않으므로 프로젝트에서 직접 구매하여 연동하셔야 합니다.
  • 다국어 지원은 어떻게 처리하나요? 
    다국어는 메시지, 날짜만 지원합니다. 그외에는 프로젝트에서 직접 구현하셔야 합니다. 메시지나 날짜 처리는 lang-XX_XX.js 파일에 정의되어 있으며,
rui_config.js에 정의된 defaultLocale 정보로 해당 국가의 메시지 처리가 가능합니다.
  • jquery와 같이 연동하여 사용하여도 되나요? 
    jquery 및 타 프레임웤과 연동하여도 RUI는 문제가 발생하지 않습니다. 다만, 다른 프레임웤이 가지는 장점들의 대부분들을 이미 RUI에 포함하고 있으므로,
굳이 따로 타 프레임웤을 사용할 필요는 없습니다.
  • async 호출시 sync 기능을 제공하나요? 
    Rui.ajax 메소드나 LDataSet에서는 sync를 지원하지만 권장하진 않습니다. 다만 LDataSetManager에서는 sync를 지원하지 않습니다.
기본은 async로 처리됩니다.
  • 라이선스 발급은 어떻게 하나요? 
    프로젝트에서 사용하는 도메일을 메일로 보내주시면 해당 프로젝트의 라이선스를 발급해드립니다.
예) 127.0.0.1;localhost;www.lgcns.com
  • 스크립트 암호화가 가능한가요? 
    암호화 라이브러리는 지원하지 않지만 표준 스크립트 방식이므로 프로젝트에서 암호화 모듈을 구입하여 적용하시면 됩니다.
  • 가우스나 MiPlatform으로 만든 화면을 RUI로 대체해서 만들 수 있나요? 
    가우스나 MiPlatform등의 콤포넌트들은 대부분 지원하므로 프로젝트에서 사용하고 있는 기능들은 90% 정도 지원하며, 
그외 Javascript framework에서 지원하는 웹 표준 기능들도 지원합니다.
