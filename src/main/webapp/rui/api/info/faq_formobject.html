Form Object

  • 자동완성 기능은 제공하나요? 
    LTextBox에서 자동 완성 기능을 이용할 수 있습니다.
textboxRemoteSample.html 샘플 참조
  • 콤보에 정의된 dataSet의 field명을 바꾸고 싶습니다. 
    콤보에는 valueField와 displayField가 정의되어 있습니다. 서버에서 전송된 데이터의 키값으로
위 속성들을 다시 정의하면 해당 변경이 가능합니다. 
다만, 콤보의 데이터를 선택하여도 선택이 되지 않을 경우에는 콤보에서 정의된 valueField값과
실제 로딩된 데이터의 키값이 맞지 않을 경우입니다. 로딩된 키값과 valueField의 값이 정확하게 
맞는지 확인하세요.
  • 콤보의 코드값만으로 콤보의 출력값을 그리드에 출력하려면 어떻게 해야 하나요? 
    그리드에서 편집 기능을 이용할 경우 콤보의 displayField값을 출력하려면 해당 LCombo의 인스턴스 객체를
LDataSet에 addTrigger 메소드와 rendererField 속성을 이용하여 자동으로 맵핑되게 연결할 수 있습니다.
  • LDateBox에서 다국어 지원은 어떻게 하나요? 
    LDateBox에 localeMask 속성이 true로 정의되면 자동으로 다국어 처리가 됩니다.
이 속성은 rui_config.js에 dateBox 항목에 기본 설정되어 있습니다.
  • 조회된 콤보의 데이터를 변경할 수 있나요? 
    LCombo의 인스턴스에 getDataSet() 메소드가 있습니다. 해당 LDataSet을 얻어와 load/add/setNameValue/remove 등을 통해서 
데이터를 변경하시면 해당 콤보의 데이터도 변경됩니다.
  • disable을 할 경우 css 제어는 어떻게 하나요? 
    L-disable css를 프로젝트에 선언한 후 아래의 값을 프로젝트에 맞게 정의 하시면 됩니다.

.L-disabled {
    color: gray;
    cursor: default;
    opacity: .6;
    -moz-opacity: 0.6;
    filter: alpha(opacity=60);
}
