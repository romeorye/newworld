다국어 설정 관련 파일은 lang-XX_XX.js에 정의되어 있다. strftime은 http://www.opengroup.org/onlinepubs/007908799/xsh/strftime.html에 오픈 그룹에 의해 정의된 여러가지 format 지정자들을 가지고 있다. PHP는 http://www.php.net/strftime에 정의된 자체의 몇가지 항목들을 추가한다. 이러한 자바스크립트 구현은 모든 PHP 지정자와 몇가지를 더 지원한다. %a - 현재 locale의 요일의 단축표시 ex) ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'] %A - 현재 locale의 요일 표시 ex) ['Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday'] %b - 현재 locale의 달의 단축표시 ex) ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'] %B - 현재 locale의 달 표시 ex) ['January', 'February', 'March', 'April', 'May', 'June', 'July', 'August', 'September', 'October', 'November', 'December'] %c - 현재 locale의 선호되는 날짜와 시간 표시 ex) 미국 : %a %d %b %Y %T %Z, 한국 : %Y년 %b %d %a %T %Z %C - century number 현재 년도를 100으로 나누고 정수로 만든값으로 00~99 %d - 달의 일을 표시하는 값으로 01 ~ 31을 표시 %D - %m/%d/%y와 같다. %e - %d와 비슷하나 1자리수의 경우 0대신이 공백이 들어간다. ' 1' ~ '31' %F - %Y-%m-%d와 같다. LDateLocale class는 Rui.util.LDate에 의해 사용되는 모든 localised date 문자열을 위한 컨테이너이며 base 클래스 이다. 이것은 내부적으로 사용되지만, 새로운 date localisation을 제공하기 위해 확장될 수 있다. 자신의 LDateLocale 생성하기 위해서, 다음 과정을 따른다:
  1. 요구와 밀접하게 일치하는 기존의 locale을 찾는다.
  2. 이것을 base class로 사용한다. 만약 맞지 않는다면, Rui.util.LDateLocale을 사용한다.
  3. Rui.merge를 사용하는 base class의 확장으로서 자신의 class를 생성하고 필요한 곳에 자신의 localisation들을 추가한다.
Rui.util.LDateLocale['en'] 부터 확장된, Rui.util.LDateLocale['en_US']와 Rui.util.LDateLocale['en_GB']을 참고한다. 예를 들어, French 프랑스어와 Canadian 프랑스어를 위한 locale들을 구현하려면, 다음 작업을 수행한다:
  1. French 프랑스어를 위해서는 기존의 유사한 locale이 없으므로 기본으로 Rui.util.LDateLocale 사용하여, 그것을 확장한다: 
         Rui.util.LDateLocale['fr'] = Rui.merge(Rui.util.LDateLocale, {
         a: ['dim', 'lun', 'mar', 'mer', 'jeu', 'ven', 'sam'],
         A: ['dimanche', 'lundi', 'mardi', 'mercredi', 'jeudi', 'vendredi', 'samedi'],
         b: ['jan', 'fév', 'mar', 'avr', 'mai', 'jun', 'jui', 'aoû', 'sep', 'oct', 'nov', 'déc'],
         B: ['janvier', 'février', 'mars', 'avril', 'mai', 'juin', 'juillet', 'août', 'septembre', 'octobre', 'novembre', 'décembre'],
         c: '%a %d %b %Y %T %Z',
         p: ['', ''],
         P: ['', ''],
         x: '%d.%m.%Y',
         X: '%T'
     });
  2. Canadian 프랑스어를 위해서는 French 프랑스어로 시작해서 \%x의 의미를 변경한다. 
         Rui.util.LDateLocale['fr-CA'] = Rui.merge(Rui.util.LDateLocale['fr'], {
         x: '%Y-%m-%d'
     });
이렇게 하면, 새로운 locale들을 사용할 수 있다:
sample: Date형을 fr 국가형 문자로 변경한다.
var d = new Date(2008, 3, 22); d.format({format: "%Y/%m/%d"}, "fr");
sample: Date형을 fr-CA 국가형 문자로 변경한다.
var d = new Date(2008, 3, 22); d.format({format: "%Y/%m/%d"}, "fr-CA");
