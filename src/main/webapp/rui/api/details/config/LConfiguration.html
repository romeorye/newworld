rui_config.js 파일의 정보를 읽어오거나 변경할 수 있는 객체 rui_config.js의 값은 json형식의 객체로 구성되어 있으며 값은 기본적으로 배열의 object형식으로 되어 있다. 값을 변경할 경우에는 내부적으로 statechange 이벤트가 발생하여 값이 바뀌는 부분에 대한 다른 처리를 할 수 있다. rui_config.js는 자바에서의 xpath와 비슷한 jsonpath로 되어 있다. set 메소드로 값을 변경할 경우에는 반드시 배열 구조의 값으로 처리되어야 한다.
sample: rui_config.js에 있는 defaultLocale정보를 얻어온다.
var defaultLocale = Rui.getConfig().get("$.core.defaultLocale")[0]; var defaultLocale = Rui.getConfig().getFirst("$.core.defaultLocale"); Rui.getConfig().set('$.core.defaultLocale', ['en_US']);
