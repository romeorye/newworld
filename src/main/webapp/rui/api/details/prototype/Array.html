prototype에 추가된 메소드 외에도 Rui.util.LArray객체를 이용하여 더 많은 기능을 사용할 수 있다. /rui/sample/general/base/prototypeSample.html
배열에 해당 값의 index를 리턴한다. 없을 경우 -1을 리턴한다.
sample: 배열에 6인 값의 index를 리턴한다.
var datas = [10, 5, 6, 4, 8]; var idx = datas.contains(6);
배열에 index의 삭제한 배열을 리턴한다.
sample: 배열에 2째 index를 제외한 배열을 리턴한다.
var datas = [10, 5, 6, 4, 8]; var newDatas = datas.removeAt(2);
