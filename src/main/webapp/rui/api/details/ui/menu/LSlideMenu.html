마우스 클릭시 아래로 슬라이딩 하며 펼쳐저 나열되는 모양의 menu를 생성한다. 샘플 경로 : /rui/sample/general/ui/menu/slideMenuSample.html
sample: 기본적인 형태의 메뉴구성 샘플
var menu = new Rui.ui.menu.LSlideMenu({ onlyOneTopOpen : false, dataSet: dataSet, fields: { rootValue: null , parentId: 'parentMenuId' , id: 'menuId' , label: 'name' , order:'seq' } }); menu.on("nodeClick", function(e) { var url = e.node.getRecord().get('url'); var depth = e.node.getDepth(); var hasChild = e.node.hasChild(); console.log('url : ' + url + ', depth : ' + depth + ', hasChild? : ' + hasChild); }); menu.render("menuDiv");
