var textBox = new Rui.ui.form.LCombo({ id:'combo1', applyTo:'combo1', url: 'test.rui', forceSelection: true });
데이터 로딩시 combo의 row 위치 기본값은 -1이다. defaultValue 속성이 우선으로 적용되고 그후에 selectedIndex가 적용된다.
sample: 데이터를 로딩한 후 기본으로 2번째 index로 셋팅한다.
var textBox = new Rui.ui.form.LCombo({ id:'combo1', applyTo:'combo1', url: 'test.rui', selectedIndex: 2 });
bind시에 현재 combo의 displayField와 맵핑된 dataSet에 반영할 출력 필드 combo 자체만으로 의미가 없고 bind시에만 의미가 있다. combo에서 값이 바뀌면 bind를 통해 code값만 반영되므로 그리드에서 출력할 display값은 변경이 안된다. 그래서 code값이 바뀌면 combo에 해당 되는 value값을 bind된 데이터의 display값에 반영할 수 있게 지원한다. rendererField는 dataSet의 addTrigger 메소드를 사용해야 한다.
sample: 데이터를 로딩한 후 기본으로 2번째 index로 셋팅한다.
var combo1 = new Rui.ui.form.LCombo({ id:'combo1', applyTo:'combo1', url: 'test.rui', rendererField: 'col1Value' }); var dataSet = new Rui.data.LJsonDataSet({ fields: [ { id: 'col1' }, { id: 'col1Value' }, ] }); dataSet.addTrigger(combo1); var bind = new Rui.data.LBind({ groupId: 'detailId', dataSet: dataSet, bindInfo: [ {id:'col1', ctrlId:'combo1', value:'value'} ] });

LCombo 생성시 초기 데이터를 로딩한다.
sample: LCombo 생성시 Y, N을 선택할 수 있는 데이터를 미리 로딩한다.
var combo = new Rui.ui.form.LCombo({ id: 'combo1', applyTo: 'combo1', items: [ { code: 'Y' }, { code: 'N' } ] });
sample: LCombo 생성시 items의 Y는 예로 표시하고 N은 아니오로 표시하는 데이터를 미리 로딩한다.
var combo = new Rui.ui.form.LCombo({ id: 'combo1', applyTo: 'combo1', items: [ { code: 'Y', value: '예' }, { code: 'N', value: '아니오' } ] });
sample: LCombo 생성시 items의 code와 value의 필드명을 변경하여 로딩한다.
var combo = new Rui.ui.form.LCombo({ id: 'combo1', applyTo: 'combo1', valueField: 'code1', displayField: 'code2', items: [ { code1: 'Y', code2: '예' }, { code1: 'N', code2: '아니오' } ] });
그리드의 데이터에는 code값만 정의되어 있고 LCombo에 display값이 있다면 code값에 대한 display값을 그리드의 셀에 출력해야 한다. autoMapping을 적용하면 그리드의 code값에 대한 display값이 출력된다.
sample: autoMapping을 적용하는 기본 샘플
var combo = new Rui.ui.form.LCombo({ autoMapping: true, applyTo: 'combo1' });
