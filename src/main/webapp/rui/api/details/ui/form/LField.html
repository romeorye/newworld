LField 객체는 form 입력 객체들을 추상 클래스 샘플 경로 : /rui/sample/general/ui/grid/gridpanelEditSample.html
field의 name으로 input, select태그들의 name 속성 값
sample: col1 콤보 객체를 dataSet의 code, value 필드와 바인드하고 col2 spane객체를 innerHTMl로 출력한다.
var textBox = new Rui.ui.form.LTextBox({ id: 'textBox1', name: 'col1' });
changed 메소드가 호출되면 수행하는 이벤트
sample: textBox1에 changed 이벤트가 발생하면 메시지를 출력한다.
var textBox1 = new Rui.ui.form.LTextBox({ id: 'textBox1', applyTo: 'col1' }); textBox1.on('changed', function(e) { alert('값이 변경되었습니다.'); });
valid 메소드가 호출되면 수행하는 이벤트
sample: textBox1에 valid 이벤트가 발생하면 L-valid css를 삭제한다.
var textBox1 = new Rui.ui.form.LTextBox({ id: 'textBox1', applyTo: 'col1' }); textBox1.on('valid', function(e) { this.removeClass('L-valid'); });
invalid 메소드가 호출되면 수행하는 이벤트
sample: textBox1에 invalid 이벤트가 발생하면 L-valid css를 추가한다.
var textBox1 = new Rui.ui.form.LTextBox({ id: 'textBox1', applyTo: 'col1' }); textBox1.on('invalid', function(e) { this.addClass('L-valid'); });
specialkey 메소드가 호출되면 수행하는 이벤트 specialkey 종류 : ctrlKey, Rui.util.LKeyListener.KEY의 모든 값, Rui.util.LKeyListener.KEY객체 참조
sample: textBox1에 specialkey 이벤트가 발생하면 ctrl키와 a키 모두 선택되었는지 확인한다.
var textBox1 = new Rui.ui.form.LTextBox({ id: 'textBox1', applyTo: 'col1' }); textBox1.on('specialkey', function(e) { if(e.keyCode == 97) { .................. } });
