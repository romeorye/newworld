LTextBox를 사용하는 기능으로 단순한 input 태그로 구현이 가능하면 LTextBox를 사용하지 말고 input태그만으로 구현이 불가능한(mask, placeholder 등...) 기능을 사용하거나 그리드에 올라가는 편집기일 경우에만 사용합니다. 샘플 경로 : /rui/sample/general/ui/form/textboxSample.html /rui/sample/general/ui/form/textboxFixedListFilterSample.html /rui/sample/general/ui/form/textboxFixedListFilterSample.html
sample1: 기본 샘플
var textBox = new Rui.ui.form.LTextBox({ applyTo: 'textBox', placeholder: 'placeholder 내용이 출력됩니다' });
input type의 종류를 설정한다. text, password, email, url 샘플 경로 : /rui/sample/general/ui/form/textboxSample.html /rui/sample/general/ui/form/textboxMaskSample.html /rui/sample/general/ui/form/textboxRemoteSample.html
sample: 기본적인 textbox를 생성한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', type: 'password' });
input박스에 키 입력이 가능한지 여부, 변경이 불가능한 읽기 전용 속성은 LUIComponent에 있는 disabled로 처리 editable이 false일 경우 setEditable 메소드를 통해 변경할 수 있다.
sample: textBox에 입력을 막는다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', editable: false });
객체에 기본적으로 들어 있을 값을 설정한다. 콤보의 경우에는 값에 해당되는 데이터가 없을 경우 빈값으로 처리된다.
sample: textBox에 기본값을 적용한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', defaultValue: '값1' });
자동 완성 기능 사용 여부 isAutoComplate가 true이면 useDataSet값은 자동으로 true로 설정된다.
sample: 이미 메모리에 로드된 자동완성기능을 이용하여 필터링한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', autoComplete : true, filterUrl: 'test.rui', filterMode: 'local' });
sample: 데이터셋을 인수로 받아서 자동완성기능을 이용하여 필터링한다.
var dataSet = new Rui.data.LJsonDataSet({ id: 'dataSet', fields: [ { id: 'code' }, { id: 'value' } ] }); var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', autoComplete : true, filterMode: 'local' });
sample: filterMode의 'remote'로 설정하여 서버에서 데이터를 가져와 필터링한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', autoComplete : true, filterMode: 'remote', filterUrl: 'test.rui' });
키보드 입력 제한을 하는 입력 가능한 문자열를 설정한다. allowChars 값에 설정된 문자만 입력 가능하게 지정한다. 단 한글은 입력이 가능하므로 imeMode를 true로 설정해야 한다. 정규식을 차후 지원 예정. 현재는 지원 안된다.
sample: textBox에 abc 키 입력만 허용하고 한글 입력을 막는다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', allowChars: 'abc', imeMode: true });
sample: textBox에 영문키과 숫자 키만 입력만 허용하고 한글 입력은 막는다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', allowChars: 'abcdefghijklmnopqrstuvwxyz1234567890', imeMode: true });
기본 '선택하세요.' 메시지 값을 설정한다. placeholder와는 다른값이다. placeholder는 focus가 되지 전에 출력할 값으로 선택 자체가 불가능하지만 emptyText는 선택이 가능하다.
sample: textBox의 선택하지 않은 값을 '전체'라는 항목으로 설정한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', emptyText: '전체' });
sample: textBox의 선택하지 않은 값을 '전체'라는 항목을 다국어로 설정한다. MessageManager의 alltext는 프로젝트에서 추가해야 한다. 없으면 에러가 발생한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', emptyText: Rui.getMessageManager().get('$.message.alltext'); });
지정된 문자열 형식만 입력하도록 한다. inputType 형식 ( 기본값은 null ) Rui.util.LString.PATTERN_TYPE_NUMBER : 숫자만 입력 Rui.util.LString.PATTERN_TYPE_STRING : 영문자만 입력 Rui.util.LString.PATTERN_TYPE_NUMSTRING : 영문자와 숫자만 입력 Rui.util.LString.PATTERN_TYPE_KOREAN : 한글만 입력
sample: textBox에 한글만 입력하도록 한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', inputType: Rui.util.LString.PATTERN_TYPE_KOREAN });
inputType에서 지정한 문자열만 입력함과 동시에 예외적으로 해당 지정 문자는 입력이 허용되도록 설정한다. inputChar 형식 ( 기본값은 null ) inputChar: '!@#'
sample: textBox에 특정문자는 입력이 허용되도록 한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', inputType: Rui.util.LString.PATTERN_TYPE_KOREAN, inputChar: '!@#' });
키 입력 mask 적용를 적용한다. mask는 자리수가 고정되어 있는 항목에만 사용 가능하다. 예를 들어 전화번호 같은 항목의 경우 국번같이 3자리나 4자리로 변경이 일정치 않은 케이스에서는 사용이 불가능하다. mask 항목 '9': "[0-9]", 'a': "[A-Za-z]", '*': "[A-Za-z0-9]"
sample: textBox에 주민등록번호와 같은 포멧을 지정한다. 999999-9999999
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', mask: '999999-9999999' });
sample: textBox에 우편번호와 같은 포멧을 지정한다. 999-999
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', mask: '999-999' });
sample: textBox에 첫번째는 영문 그뒷자리로 5자리는 숫자로 지정한다. a99999
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', mask: 'a999999' });
sample: textBox의 선택하지 않은 값을 '전체'라는 항목을 다국어로 설정한다. MessageManager의 alltext는 프로젝트에서 추가해야 한다. 없으면 에러가 발생한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', emptyText: Rui.getMessageManager().get('$.message.alltext'); });
html5에 있는 placeholder 기능 placeholder는 textbox가 focus 되지 전에 사용자에게 메시지를 전달하는 기능이다.
sample: textBox의 focus 되기전에 '이메일을 입력하세요.'라는 항목을 출력한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', placeholder: '이메일을 입력하세요.' });
값을 변경한다.
sample: textBox의 'aaa'라는 값을 반영한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1' }); textBox.setValue('aaa');
자동완성 기능을 사용할때 dataset을 리턴한다.
sample: textBox를 자동완성 기능으로 설정하고 키 입력시 데이터가 없으면 메시지를 출력한다.
var textBox = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1', isAutoComplate: true, filterUrl: 'test.rui' }); var dataSet = textBox.getDataSet(); dataSet.on('load', function(e) { if(dataSet.getCount() < 1) { alert('필터된 데이터가 한건도 없습니다.'); } });
keydown 기능이 호출되면 수행하는 이벤트. 기본적인 dom 이벤트
sample: textBox1에서 enter(13)키를 입력하면 textBox2로 이동한다.
var textBox1 = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1' }); textBox1.on('keydown', function(e) { if(e.keyCode == 13) { textBox2.focus(); } });

expand(자동완성기능이나 콤보의 목록이 펼쳐질때) 기능이 호출되면 수행하는 이벤트

collapse(자동완성기능이나 콤보의 목록이 닫힐때) 기능이 호출되면 수행하는 이벤트

var textBox1 = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1' }); textBox1.on('collapse', function(e) { if(textBox1.getValue() == '') { alert('값을 선택해야 합니다.'); } });
setValue 메소드가 호출되면 수행하는 이벤트
sample: textBox에 바뀐값이 'aaa'면 메시지를 출력하고 다시 focus한다.
var textBox1 = new Rui.ui.form.LTextBox({ id:'textbox1', applyTo:'textbox1' }); textBox1.on('changed', function(e) { if(textBox1.getValue() == 'aaa') { alert('aaa값은 입력할 수 없습니다.'); textBox1.focus(); } });
