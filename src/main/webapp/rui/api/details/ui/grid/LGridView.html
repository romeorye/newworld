기본 그리드는 buffer 방식으로 성능위주의 그리드이므로 표현에 제한이 따른다. 그러나 LGridView는 모든 row를 html dom을 생성하므로서 원하는 포멧으로 출력이 가능하다. 단 LGridView를 상속받고 있으므로 대량건 처리는 불가능하다.(ie8기준 최대 300건) 샘플 경로 : /rui/sample/general/ui/grid/gridpanelGridViewSample.html
sample: 기본 샘플이다.
var grid = new Rui.ui.grid.LGridPanel({ columnModel: columnModel, dataSet: dataSet, height: 420, view: new Rui.ui.grid.LGridView({ dataSet: dataSet, columnModel: columnModel }) });
