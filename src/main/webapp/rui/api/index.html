DevOn RichUI javascript framework API

아이콘 설명
  • 클래스 : new로 생성할 수 있는 기본 클래스
  • static 클래스 : new로 생성할 수 없고 상속도 받을 수 없는 클래스
  • 생성자 메소드 : 생성자 메소드
  • 생성자 환경정보 : new 생성자로 객체를 생성할 경우 생성자 인수로 넣을 수 있는 속성
  • 이벤트 : on으로 걸 수 있는 이벤트
  • 메소드 : 기본 메소드
  • static 메소드 : static 메소드
  • property : 클래스의 속성
  •   static : static 형태로 제공되는 요소에 붙는 접미항목
  •   plugin : plugin 형태로 제공되는 요소에 붙는 접미항목
  •   protected : protected 형태로 제공되는 요소에 붙는 접미항목 (protected는 public형태로 사용해서는 안됩니다.)
주의사항
  • 화면 갱신이 24시간이상 반복 호출되는 페이지의 경우 ajax로 처리하면 안됨
  • 화면에 콤포넌트(DataSet, Grid, Tree, Tab 등...) 갯수가 10개 이상이되면 최적화 필요 (RichUI 담당 그룹에 가이드를 요청)
  • 그리드에서 renderer로 javascript:function()과 같은 스크립트 호출시 메모리릭 발생
